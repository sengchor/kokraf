Knife Tool

The Knife Tool allows you to cut geometry manually by drawing a line across the mesh. It can create new vertices and edges by slicing through faces, edges, or snapping directly to existing vertices.

The Knife Tool works in screen space and only affects visible geometry.

How to Use

  • Activate the Knife Tool. K
  • Left Click to place the first cut point.
  • Move the mouse to preview the cut line and intersections.
  • Left Click again to place the second cut point and apply the cut.
  • Cancel the operation with Esc before confirming the second point.

Snapping

The Knife Tool supports snapping to existing vertices for precise cuts.

  • Snapping is only available in Vertex Select mode.
  • Hover near a vertex to snap the cut point to it.
  • Snapped vertices are reused and not duplicated.

Cut Behavior

  • The cut only affects visible faces facing the camera.
  • Intersections are created where the cut line crosses edges.
  • Faces intersected by the cut are automatically split into new faces.
  • If the cut follows an existing edge or polyline, no new geometry is created.

Selection After Cut

After applying a cut, the resulting selection depends on the current Edit Mode selection type.

  • Vertex Select Mode — Newly created vertices are selected.
  • Edge Select Mode — Newly created edges are selected.
  • Face Select Mode — The selection is cleared.

The Knife Tool is ideal for adding controlled topology and refining mesh detail.
