Import

The Import feature is located in the File submenu of the menubar. It allows you to bring external 3D objects into your current scene. Currently, the editor supports importing OBJ files.

How to Import

  • Click the Import button in the File menu.
  • Select a .obj file from your computer using the file picker dialog.
  • The selected object will be loaded into your scene, maintaining its geometry and position.

Notes

  • The imported object will be added to the current scene without replacing existing objects.
  • Only OBJ format is supported for now. Other formats may need conversion before import.
  • Imported objects can be edited, transformed, and exported just like objects created inside the editor.
