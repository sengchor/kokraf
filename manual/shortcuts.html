Keyboard Shortcuts

Here are some essential keyboard shortcuts to help you speed up your workflow in Kokraf.

Tool / Action Mode Shortcut
Select Tool Object / Edit W
Move Tool Object / Edit G
Rotate Tool Object / Edit R
Scale Tool Object / Edit S
Extrude Tool Edit E
LoopCut Tool Edit Ctrl + R
Knife Tool Edit K
Object Focus Object Shift + F
Orbit Camera Any Middle Mouse
Pan Camera Any Shift + Middle Mouse
Zoom Camera Any Mouse Wheel
Undo Any Ctrl + Z
Redo Any Ctrl + Shift + Z
Switch Mode Any Tab
Multi Select Any Shift + Left Mouse
New Face From Vertices Edit F
Seperate Edit P
Delete Object / Edit Delete
Duplicate Object Shift + D

Note: Some shortcuts may depend on your current selection mode (Object / Edit).
