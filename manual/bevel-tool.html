Bevel Tool

The Bevel Tool allows you to create smooth edges by cutting and expanding selected edges.

It is commonly used to round corners, add supporting edges, and improve shading.

Bevel adds new geometry along edges, creating chamfers or rounded transitions.

How to Use Bevel

  • Select edges to bevel.
  • Activate the Bevel Tool. Ctrl + B
  • Move the mouse to adjust the bevel width.
  • Confirm with Left Click or Enter.
  • Cancel with Esc.

Bevel Behavior

  • Inserts additional vertices along each selected edge.
  • Generates new faces between these vertices to replace the original sharp edge.
  • Fills corners where beveled edges meet.

Segments

  • Scroll the mouse wheel to increase or decrease the number of segments.
  • More segments create a smoother, rounded bevel.

Width Control

  • Move the mouse to control bevel width.
  • Use numeric input for precise values.

Bevel is essential for realistic models, as perfectly sharp edges rarely exist in real-world objects.
