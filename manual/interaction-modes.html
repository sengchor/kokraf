Interaction Modes

Interaction Modes define how you interact with objects in the viewport. The two main modes are Object Mode and Edit Mode.

Location

  • The Interaction Modes dropdown is located at the top-left corner of the viewport.

Object Mode

  • Object Mode allows you to select, move, rotate, and scale whole objects.
  • All objects in the scene are selectable.
  • Some tools and features are only available in Object Mode.
  • The selection box can be used to select multiple objects in Object Mode.

Edit Mode

  • Edit Mode allows you to modify the geometry of a mesh, including vertices, edges, and faces.
  • You must select exactly one mesh object before entering Edit Mode.
  • Edit Mode enables sub-selection modes: Vertex, Edge, and Face.
  • Some tools are only available in Edit Mode, such as vertex snapping, extrusion, or direct geometry manipulation.
  • You cannot select other objects while in Edit Mode; only the geometry of the selected mesh can be modified.

Notes

  • Some features are available in both Object and Edit Mode, such as transform controls (move, rotate, scale) and snapping.
  • Switching between modes automatically updates the available tools and viewport helpers.
  • Attempting to enter Edit Mode without a mesh selected will display a warning.

Using the correct interaction mode is important for efficient workflow. Object Mode is for scene-level manipulation, while Edit Mode is for detailed geometry editing.
