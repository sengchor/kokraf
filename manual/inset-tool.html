Inset Tool

The Inset Tool allows you to create a new face inside the selected faces by scaling them inward and generating supporting edges.

It is commonly used to add details, create borders, or prepare geometry for further operations like extrusion.

Inset works only on selected faces. It creates new geometry inside the original faces while keeping the outer shape intact.

How to Use Inset

  • Select one or more faces.
  • Activate the Inset Tool. I
  • Move the mouse to adjust the inset width.
  • Confirm with Left Click or Enter.
  • Cancel with Esc. The geometry will return to its original state.

Inset Behavior

  • Single Face: The face is inset uniformly toward its center, creating a smaller face inside it.
  • Multiple Faces: Connected faces are treated as a region, creating a continuous inset with new side faces around the boundary.

Thickness Control

The distance you move the mouse determines the thickness of the inset.

You can also enter a numeric value for the inset width.

Inset is often used before extrusion to create panels, frames, or detailed surface features.
