Selection

Kokraf provides 2 ways to select objects or components in your 3D scene. Understanding selection is essential for efficient modeling workflows.

Click Selection

Click selection allows you to select objects individually.

  • Left Mouse Click: select single object
  • Shift + Left Mouse Click: select multiple objects
  • Left Mouse Click empty space: deselect all

Box Selection

Box selection allows you to select multiple objects at once by dragging a rectangular area around them.

  • Drag with the Left Mouse button to create a selection box
  • All objects fully or partially within the box will be selected when you release the Left Mouse

Both selection methods work in Object Mode and Edit Mode. In Edit Mode, you can select vertices, edges, or faces.
