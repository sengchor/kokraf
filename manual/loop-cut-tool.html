Loop Cut Tool

The Loop Cut Tool allows you to insert a new edge loop into a mesh while in Edit Mode.

Loop cuts work on continuous quad-based geometry. The new edges are inserted at the midpoint of existing edges.

How to Use

  • Activate the Loop Cut Tool. Ctrl + R or click the Loop Cut tool in the viewport toolbar.
  • Hover over the mesh to preview the edge loop. A yellow highlight indicates where the cut will occur.
  • Click Left Click to apply the loop cut.

Loop Cut Behavior

  • Loop cuts follow connected quad faces to form a continuous edge loop.
  • If the loop forms a closed ring, the cut will wrap fully around the mesh.
  • If the loop is open, the cut will stop at the mesh boundary or when it encounters a non-quad face.

Selection After Loop Cut

  • After applying a loop cut, the newly created geometry is automatically selected.
  • In Vertex Select Mode, the new vertices created by the loop cut are selected.
  • In Edge Select Mode, the new edge loop is selected.
  • In Face Select Mode, no faces are selected.

This allows you to immediately move, scale, or slide the new loop without reselecting it.
