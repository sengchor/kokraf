X-Ray Mode

X-Ray Mode allows you to see through objects in the scene, making it easier to select or edit geometry that is hidden behind other objects.

Location

  • The X-Ray toggle is located at the top-right corner of the viewport, next to the viewport shading controls.
  • Click the X-Ray icon to enable or disable the mode.

Functionality

  • When X-Ray is enabled, all objects in the scene become partially transparent.
  • This allows you to select vertices, edges, and faces that are normally hidden behind other geometry.
  • X-Ray works in combination with the selection tools, including click and box selection.
  • It can be used together with snapping and transform tools for precise editing of hidden components.

Shading Compatibility

  • X-Ray mode works only with Solid and Normal shading.
  • It will not work with Material or Wireframe shading.
  • If you switch to a shading mode where X-Ray does not work, it will automatically disable.
  • When you switch back to a compatible shading mode (Solid or Normal), X-Ray will automatically re-enable.

Notes

X-Ray does not affect the final render; it is only for viewport editing and selection.

Enabling X-Ray is especially helpful in Edit Mode when working on internal components of a mesh.
