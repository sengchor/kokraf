Export

The Export feature allows you to save your 3D objects into various file formats for use in other 3D software, game engines, or for sharing.

Location

  • The Export option is located in the File > Export submenu of the editor.

Supported Formats

  • GLB (.glb) — Binary GLTF format, compact and widely supported.
  • GLTF (.gltf) — JSON-based GLTF format, readable and editable.
  • OBJ (.obj) — Geometry format including vertices, normals, and faces.
  • STL (.stl) — Plain-text STL format for 3D printing.
  • Binary STL (.stl-binary) — Binary STL format for more compact storage.
  • USDZ (.usdz) — Apple AR format for ARKit and USDZ viewers.

Recommended Format

For preserving your original polygon geometry and topology, it is recommended to export using the OBJ (.obj) format. Other formats (GLB, GLTF, STL, USDZ) will automatically triangulate your mesh during export, which can make it harder to edit or modify later.

Requirements

  • You must be logged in to export objects. If not logged in, the editor will prompt you to log in.
  • Exports may require credits if your account has usage limitations. You will be notified if you cannot export.

Export Behavior

  • Multiple objects can be exported simultaneously. All objects are merged into a single file while preserving transformations.
  • Geometry is processed according to its shading mode (smooth, flat, or auto), ensuring normals and faces are correctly exported.
  • World transformations (position, rotation, scale) are applied to the exported geometry.

How to Export

  • Go to File > Export and select the desired format.
  • Ensure the objects you want to export are selected.
  • The file will automatically be downloaded once exported.
  • If the export fails, check your login status, available credits, and that the selected objects are valid meshes.

Exporting allows you to share your 3D creations or import them into other applications while preserving geometry, shading, and transformations. For best results, always ensure your objects are properly transformed and have the correct shading applied before exporting.
