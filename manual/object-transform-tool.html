Object Transform Tool

The Object Transform Tool allows you to move, rotate, and scale objects in the scene while in Object Mode.

Before transforming, you need to select the objects you want to modify.

Object transformations affect the entire object, not individual vertices, edges, or faces.

Move (Translate)

Move changes the position of the selected object in 3D space.

  • Activate Move. G
  • Move the mouse to reposition the object.
  • Confirm with Left Click or Enter.
  • Cancel with Esc.

Rotate

Rotate spins the selected object around its origin.

  • Activate Rotate. R
  • Move the mouse to rotate the object.
  • Confirm with Left Click or Enter.
  • Cancel with Esc.

Scale

Scale changes the size of the selected object.

  • Activate Scale. S
  • Move the mouse to resize the object.
  • Confirm with Left Click or Enter.
  • Cancel with Esc.

Transform Gizmo

In addition to keyboard shortcuts, you can control object transformations using the Transform Gizmo.

The gizmo appears on the selected object and provides visual handles for moving, rotating, and scaling.

  • Click a gizmo handle to select an axis.
  • Drag the mouse to apply the transformation along that axis.
  • Release the mouse button to confirm the transformation.

Gizmo interactions allow precise, visual control and are especially useful for quick adjustments.

Axis Constraints

Transformations can be constrained to a specific axis for precise control.

  • Press X, Y, or Z after starting a transform to lock movement to that axis.

Axis constraints work with Move, Rotate, Scale, and the Transform Gizmo.
