Kokraf — Features

Professional 3D Modeling, Directly in Your Browser

Kokraf is a web-based 3D modeling and geometry editing tool built for game developers, creators, and technical artists. No installation required — create, edit, and export 3D assets directly from your browser.

Core Features

Browser-Based 3D Editor

Create and edit 3D models directly in your browser with a real-time viewport.

Geometry Editing Tools

Move, rotate, and scale geometry with tools that support polygon faces.

Import 3D Models

Import models into the application and start editing or integrating them into your projects instantly.

Export for Production

Export models in .glb, .obj, .stl, and .usdz formats, ready for game engines.
